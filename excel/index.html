Afbeeldingen in Excel

Excel is een van de meest toeganglijk, multifunctionele programma's. Het kan voor bijna alles gebruikt worden, en zo ook het laten zien van afbeeldingen in cellen. Elke afbeeldinge bestaad uit pixels, die vervolgens weet uit drie waardes bestaan tussen de 0 en 255: rood, groen en blauw. 0 is niks, zwart en 255 is zo fel mogelijk rood, groen of blauw. Deze getallen kan je in een excelsheet zetten met conditionele opmaak die precies hetzelfde doet als hoe een afbeelding wordt laten zien door de computer. De conditionele opmaak maak 0 zwart, en 255 zo fel nogelijk. Hierdoor kan je de afbeelding zien in door alleen naar gekleurde cellen te kijken.

from PIL import Image
import xlsxwriter

im = Image.open('afbeelding.jpg')
pix = im.load()
workbook = xlsxwriter.Workbook('resultaat.xlsx')
worksheet = workbook.add_worksheet()

for x in range(im.size[1]):
 for y in range(im.size[0]):
  worksheet.write(x, y*3, pix[y,x][0])
  worksheet.write(x, y*3+1, pix[y,x][1])
 worksheet.write(x, y*3+2, pix[y,x][2])

for y in range(im.size[1]):
 worksheet.conditional_format(0,y*3+0,im.size[1]+4,y*3, {'type':'2_color_scale', 'min_color': "#000000",'max_color': "#FF0000"})
 worksheet.conditional_format(0,y*3+1,im.size[1]+4,y*3+1, {'type':'2_color_scale','min_color': "#000000",'max_color': "#00FF00"})
 worksheet.conditional_format(0,y*3+2,im.size[1]+4,y*3+2, {'type':'2_color_scale','min_color': "#000000",'max_color': "#0000FF"})

for x in range(im.size[0]*3):
 worksheet.write(im.size[1]+1,x, 0)
 worksheet.write(im.size[1]+2,x, 255)

workbook.close()
